# 3DeSktop
3DeSktop is a 3DS homebrew app to mirror your computer's screen to your 3DS.  
You can find a macOS streamer at [jcrm1/threedsmirror](https://github.com/jcrm1/threedsmirror).  
Uses [QOI](https://qoiformat.org).
